# Integration: WeCom

WeCom (企业微信) SSO, contacts sync, and application message push.

## Features

- **OAuth SSO**: Green branded login button, auth type `wecom-oauth`
- **Contacts Sync**: Department + user sync via WeCom API
- **Message Push**: Application messages via corp conversation API
- **Admin Config**: Corp ID, Corp Secret, Agent ID

## Setup

1. Create an application in WeCom admin console
2. Configure Corp ID, Secret, Agent ID in plugin settings
3. Enable the `wecom-oauth` authenticator
