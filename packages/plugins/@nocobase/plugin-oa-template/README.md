# OA Template

Pre-built OA collections for office automation.

## Collections

| Collection | Fields | Purpose |
|-----------|--------|---------|

| `oaMeetingRooms` | 6 | Meeting room inventory |
| `oaMeetingBookings` | 9 | Room bookings with recurring support |
