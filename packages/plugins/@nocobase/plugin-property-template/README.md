# Property Management Template

Pre-built collections for property/community management.

## Collections

| Collection | Fields | Purpose |
|-----------|--------|---------|
| `propOwners` | 7 | Owner/tenant profiles with unit and building |
| `propRepairRequests` | 10 | Repair requests with category, priority, cost |
| `propFees` | 8 | Property fees with type, period, payment tracking |
