# Integration: Feishu / Lark

Feishu (飞书) SSO, contacts sync, and message push.

## Features

- **OAuth SSO**: Login button, auth type `feishu-oauth`
- **Contacts Sync**: Department + user sync via Feishu Open API
- **Message Push**: Send messages via IM API
- **Admin Config**: App ID, App Secret

## Setup

1. Create an app at [open.feishu.cn](https://open.feishu.cn)
2. Configure App ID and Secret in plugin settings
3. Enable the `feishu-oauth` authenticator
