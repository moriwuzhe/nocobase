# Template Market

Browse and activate pre-built industry/business templates with one click.

## Features

- **Visual catalog**: Card-based grid with icons, descriptions, and category filtering
- **Category filter**: Sales, Supply Chain, Finance, HR, Office, Service, Assets, Community
- **Search**: Full-text search across template names and tags
- **One-click activate/deactivate**: Enable templates via plugin manager
- **Detail view**: See included collections and plugin info before activating
- **Status tracking**: Visual badges for activated templates

## API

| Endpoint | Description |
|----------|-------------|
| `templateMarket:list` | List all templates with status |
| `templateMarket:getDetail` | Get template details |
| `templateMarket:activate` | Enable a template plugin |
| `templateMarket:deactivate` | Disable a template plugin |
