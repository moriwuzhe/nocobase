# Education Template

Pre-built collections for education management.

## Collections

| Collection | Fields | Purpose |
|-----------|--------|---------|
| `eduStudents` | 10 | Student profiles with grade, class, parent info |
| `eduCourses` | 8 | Course catalog with teacher, credits, schedule |
| `eduGrades` | 6 | Student grades (score + letter grade + comment) |
