# NocoBase 超越明道云和敲敲云的全面提升计划

## 一、当前项目状况分析

### 1.1 项目概览

NocoBase v1.9.47 是一个基于插件化微内核架构的开源无代码/低代码平台，当前拥有：
- **核心模块**: 22 个（ACL、数据库、客户端、服务器、缓存、日志等）
- **插件数量**: 83 个
- **代码规模**: 5,650+ TypeScript/TSX 文件
- **技术栈**: React 18 + Ant Design 5 + Formily + Sequelize + Koa

### 1.2 现有功能清单

| 分类 | 现有功能 |
|------|---------|
| 数据模型 | 主数据源、外部数据源、SQL集合、树形集合、图形化集合管理 |
| 字段类型 | 公式字段、序列号、排序字段、多对多数组、中国地区、附件URL、Markdown、快照字段 |
| 区块/视图 | 表格、表单、详情、日历、甘特图、看板、网格卡片、列表、地图、iframe、数据可视化 |
| 操作 | 批量编辑、批量更新、自定义请求、复制、导出、导入、打印、文本复制 |
| 工作流 | 集合触发器、定时触发器、操作触发器 + 计算/条件/创建/删除/查询/更新/结束 + 聚合/抄送/延迟/动态计算/JSON查询/循环/邮件/手动/通知/并行/HTTP请求/SQL |
| AI | AI管理器、OpenAI/DeepSeek提供者、LLM工作流节点 |
| 权限 | ACL、角色策略、资源级权限、代码片段权限 |
| 认证 | 基础认证、短信认证、API Keys |
| 通知 | 通知管理器、邮件通知、应用内消息 |
| 系统 | 主题编辑器、备份恢复、环境变量、日志、文件管理、国际化、多应用、公开表单、API文档 |
| 移动端 | 移动端适配插件（基础） |

## 二、竞品对比分析

### 2.1 明道云 (HAP) 核心优势

明道云是国内领先的APaaS平台，其核心优势包括：

| 功能领域 | 明道云具备 | NocoBase 现状 | 差距等级 |
|---------|-----------|-------------|---------|
| **审批流程** | 完整的审批引擎（多级审批、会签、或签、加签、转签） | 仅有手动节点，无专业审批引擎 | 🔴 高 |
| **子流程/流程编排** | 子工作流、流程嵌套、流程版本管理 | 无子流程、版本管理简单 | 🔴 高 |
| **门户/外部用户** | 外部门户、外部用户管理、外部分享 | 仅有公开表单 | 🔴 高 |
| **讨论/评论系统** | 记录级讨论、@提及、评论流 | 无 | 🔴 高 |
| **高级报表** | 透视表、数据透视图、复杂图表组合、报表导出 | 基础图表（data-visualization） | 🟡 中 |
| **企业集成** | 企业微信、钉钉、飞书、微信支付等深度集成 | 无国内企业平台集成 | 🔴 高 |
| **数据行级权限** | 基于条件的行级数据权限 | ACL资源级权限 | 🟡 中 |
| **字段级权限** | 字段级读/写权限控制 | 有基础实现 | 🟡 中 |
| **OCR/文字识别** | 内置OCR识别能力 | 无 | 🟡 中 |
| **电子签名** | 电子签名功能 | 无 | 🟡 中 |
| **高级表单联动** | 丰富的表单联动、级联规则 | 基础联动 | 🟡 中 |
| **打印模板引擎** | Word/Excel/PDF模板，模板设计器 | 基础打印 | 🟡 中 |
| **数据工厂/ETL** | 数据清洗、转换、同步 | 无 | 🟡 中 |
| **应用市场/模板** | 丰富的应用模板、行业方案 | 无应用模板市场 | 🔴 高 |
| **统计/汇总字段** | 高级统计/汇总/计算字段 | 公式字段（基础） | 🟡 中 |
| **Webhook高级** | 双向Webhook、事件订阅 | 基础自定义请求 | 🟡 中 |
| **移动端体验** | 成熟的移动端App、扫码、定位等 | 基础移动适配 | 🔴 高 |
| **数据关联图谱** | 关联关系可视化 | 图形化集合管理（基础） | 🟡 中 |

## 三、超越计划：分阶段实施路线图

### 第一阶段：基础能力补齐（P0 - 优先级最高）

> 目标：补齐明道云和敲敲云的核心差距功能，让平台具备企业级应用的基础能力

#### 1.1 专业审批引擎 `plugin-workflow-approval`

**目标**: 构建完整的审批流程引擎，超越明道云的审批能力

```
packages/plugins/@nocobase/plugin-workflow-approval/
├── src/
│   ├── server/
│   │   ├── ApprovalInstruction.ts        # 审批节点指令
│   │   ├── ApprovalCenterService.ts      # 审批中心服务
│   │   ├── ApprovalDelegation.ts         # 审批委托
│   │   ├── models/
│   │   │   ├── ApprovalRecord.ts         # 审批记录
│   │   │   └── ApprovalComment.ts        # 审批意见
│   │   └── strategies/
│   │       ├── SequentialStrategy.ts     # 依次审批
│   │       ├── CountersignStrategy.ts    # 会签（全部通过）
│   │       ├── OrSignStrategy.ts         # 或签（一人通过即可）
│   │       ├── VoteStrategy.ts           # 票签（按比例）
│   │       └── CustomStrategy.ts         # 自定义策略
│   └── client/
│       ├── ApprovalNode.tsx              # 审批节点UI
│       ├── ApprovalCenter.tsx            # 我的审批中心
│       ├── ApprovalTimeline.tsx          # 审批时间线
│       └── ApprovalActions.tsx           # 审批操作（同意/拒绝/转签/加签/退回）
```

**核心功能**:
- 多级审批链（支持无限层级）
- 会签（AND逻辑）、或签（OR逻辑）、票签（比例投票）
- 加签（动态添加审批人）、转签（转交他人审批）
- 审批委托（出差/请假时自动委托）
- 审批退回（退回任意节点/退回上一节点/退回发起人）
- 催办功能（超时自动催办、手动催办）
- 审批意见（带附件的审批意见）
- 审批中心（待审批、已审批、我发起的）
- 审批统计（审批时效、审批通过率等）
- 移动端审批（推送通知+移动端审批）

## 五、核心超越策略

### 5.1 NocoBase 的独特优势（要继续放大）

1. **数据模型驱动** → 继续深化，这是比明道云的表单驱动更先进的架构
2. **插件化微内核** → 保持开放性，让社区和第三方能持续贡献
3. **开源** → 利用开源优势建立开发者社区生态
4. **AI原生** → 在AI方面全面领先，这是竞品短期难以追赶的

### 5.2 差异化超越点

| 超越维度 | 策略 | 目标效果 |
|---------|------|---------|
| **AI深度融合** | AI不只是聊天，而是融入每个操作环节 | 明道云/敲敲云无此能力 |
| **插件生态** | WordPress式的插件市场，开源社区贡献 | 功能覆盖远超封闭平台 |
| **数据模型先进性** | 数据模型驱动 > 表单驱动 | 复杂业务场景更灵活 |
| **企业级开放性** | 完全可控的部署、二次开发、数据所有权 | 对数据安全要求高的企业更有吸引力 |
| **审批+BPM双引擎** | 轻量审批+专业BPM并存 | 覆盖简单到复杂全场景 |
| **移动端+AI** | 移动端AI助手、语音交互 | 移动办公体验质的飞跃 |

### 5.3 技术架构优化建议

1. **微服务化**: 核心服务可拆分为独立微服务，提升可扩展性
2. **事件驱动架构**: 引入消息队列（如 Redis Streams/RabbitMQ），解耦服务间通信
3. **实时协作**: 引入 CRDT/OT 算法，支持多人实时协作编辑
4. **缓存策略**: 多级缓存（内存→Redis→数据库），提升大数据量下的查询性能
5. **GraphQL支持**: 除REST API外，提供GraphQL接口，满足灵活查询需求

## 七、总结

本计划从6个维度系统性地规划了NocoBase超越明道云和敲敲云的路径：

1. **基础能力补齐**（审批、门户、企业集成）—— 消除核心差距
2. **能力深度增强**（报表、权限、表单、BPM）—— 达到并超越竞品水平
3. **AI深度融合**（搜索、分析、Agent、文档）—— 建立差异化壁垒
4. **行业方案与生态**（模板市场、行业模板）—— 形成平台护城河
5. **移动端与体验**（移动增强、消息中心）—— 全面提升用户体验
6. **企业基础设施**（高可用、安全、运维）—— 支撑企业级部署

NocoBase 最大的机会在于：**利用开源优势和AI原生能力，在保持架构先进性的同时，快速补齐功能短板，并在AI融合这一维度上远远甩开竞品。**
